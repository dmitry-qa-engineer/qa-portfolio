## 36-1 — Не сохраняются изменения полей "Имя" и "Занятие" после нажатия кнопки "Сохранить"

- **Приоритет:** Блокирующий
- **Статус:** Открыт
- **Предусловия:** Открыто окно редактирования профиля
- **Ожидаемый результат:** На главной странице в профиле отображается “Иван” и “Художник”
- **Фактический результат:** Данные не изменились, отображаются старые данные. Скринкаст
- **Окружение:** Windows 11, браузер Google Chrome Версия 144.0.7559.133

**Шаги воспроизведения:**
- 1. Кликнуть на строку ввода "Имя"
- 2. Ввести “Иван”
- 3. Кликнуть на строку ввода "Занятие"
- 4. Ввести “Художник”
- 5. Нажать кнопку “Сохранить”
